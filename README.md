# 医院挂号管理系统(Hospital Manager System)
这是一个基于C++开发的医院挂号管理系统，系统旨在模拟医院进行挂号，医生问诊，开单以及缴费的全流程，为患者提供便捷的挂号服务，为医生和管理者提供高效的管理平台。
##项目亮点
###角色权限管理：本系统针对用户，医生和管理员设置了不同的GUI用户界面。
###在线挂号流程：支持用户进行科室选择，医生选择以及预约挂号，以及相应的症状描述。
###在线问诊开单流程：支持医生在线进行挂号单增删改查，以及在线选择药物，开出药单以及诊断。
###数据存储设计：利用MySQL进行数据元素存储，包括科室信息，医生信息，挂号单信息以及相应的用户信息
